# Defensive Security - Exercises

Exercise 1: Threat Hunting
Analyze 24 hours of logs and identify APT activity
Look for: C2 beacons, lateral movement, credential dumping

Exercise 2: Incident Response
Respond to ransomware attack
Steps: Contain, eradicate, recover, document

Exercise 3: SIEM Rule Creation
Write detection rules for:
- PowerShell obfuscation
- Kerberoasting
- Data exfiltration

Exercise 4: Log Analysis
Given Sysmon logs, identify:
- Process injection
- Privilege escalation
- Persistence mechanisms

Exercise 5: Threat Intelligence
Correlate IOCs with MITRE ATT&CK tactics
Identify threat actor based on TTPs

Exercise 6: Behavioral Analysis
Detect anomalies in user behavior
Flag suspicious authentication patterns

Exercise 7: IR Playbook
Create playbook for phishing incident
Include detection, containment, communication

Exercise 8: Memory Forensics
Analyze memory dump for malware artifacts
Identify C2 servers and persistence
